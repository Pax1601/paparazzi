1%WORLD2CAM projects a 3D point on to the image
 2%   m=WORLD2CAM(M, ocam_model) projects a 3D point on to the
 3%   image and returns the pixel coordinates.
 4%   
 5%   M is a 3xN matrix containing the coordinates of the 3D points: M=[X;Y;Z]
 6%   "ocam_model" contains the model of the calibrated camera.
 7%   m=[rows;cols] is a 2xN matrix containing the returned rows and columns of the points after being
 8%   reproject onto the image.
 9%   
10%   Copyright (C) 2006 DAVIDE SCARAMUZZA   
11%   Author: Davide Scaramuzza - email: davide.scaramuzza@ieee.org
12
13function m=world2cam(x, y, z)
14
15M = [x; y; z];
16
17ss = [-1.342988975156848e+02;0;0.002319848981658;1.913645310011797e-05];
18xc = 1.361364707499628e+02;
19yc = 1.200594502378968e+02;
20width = 272;
21height = 272;
22c = 1.008129696791430;
23d = 0.005332214439608;
24e = 0.018209638073116;
25
26m = [0, 0];
27
28[x,y]  = omni3d2pixel(ss,M,width, height);
29m(1) = real(x*c + y*d + xc);
30m(2) = real(x*e + y   + yc);
