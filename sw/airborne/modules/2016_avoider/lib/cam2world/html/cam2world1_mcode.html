1%CAM2WORLD Project a give pixel point onto the unit sphere
 2%   M=CAM2WORLD=(m, ocam_model) returns the 3D coordinates of the vector
 3%   emanating from the single effective viewpoint on the unit sphere
 4%
 5%   m=[rows;cols] is a 2xN matrix containing the pixel coordinates of the image
 6%   points.
 7%
 8%   "ocam_model" contains the model of the calibrated camera.
 9%   
10%   M=[X;Y;Z] is a 3xN matrix with the coordinates on the unit sphere:
11%   thus, X^2 + Y^2 + Z^2 = 1
12%   
13%   Last update May 2009
14%   Copyright (C) 2006 DAVIDE SCARAMUZZA   
15%   Author: Davide Scaramuzza - email: davide.scaramuzza@ieee.org
16
17function M=cam2world(x, y)
18
19m = [x; y];
20
21n_points = 1;
22
23ss = [-1.342988975156848e+02;0;0.002319848981658;1.913645310011797e-05];
24xc = 1.361364707499628e+02;
25yc = 1.200594502378968e+02;
26width = 272;
27height = 272;
28c = 1.008129696791430;
29d = 0.005332214439608;
30e = 0.018209638073116;
31
32A = [c,d;
33     e,1];
34T = [xc;yc]*ones(1,n_points);
35
36m = A^-1*(m-T);
37M = getpoint(ss,m);
